# Lens

Real-time camera engine for iOS built with **Swift, Metal and AVFoundation**.

Lens is an experimental camera application that explores **real-time visual effects, LiDAR depth processing, and voice-controlled shader systems** running directly on the iPhone GPU.

The project focuses on building a **high-performance rendering pipeline** capable of maintaining **60 FPS real-time graphics** while applying multiple GPU effects to the live camera feed.

---

## Features

• Real-time **Metal shader engine**  
• **30+ visual effects** implemented with GPU shaders  
• **LiDAR depth-aware effects**  
• **Voice-controlled effect system (speech-to-effect)**  
• Custom **camera pipeline built on AVFoundation**  
• Dynamic **FPS optimization system**  
• Modular **shader architecture** for adding new effects  

---

## Technologies

• Swift  
• Metal (GPU shaders)  
• AVFoundation camera pipeline  
• LiDAR depth processing  
• SwiftUI interface  
• Custom render pipeline architecture  

---

## Project Goal

Lens is built as a **research and portfolio project** exploring advanced real-time graphics techniques on mobile devices, including shader pipelines, camera processing, and interactive visual systems.

The goal is to demonstrate how modern iPhones can act as **real-time GPU visual engines**, similar to tools like TouchDesigner or Resolume, but running natively on mobile hardware.
